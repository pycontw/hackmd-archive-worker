# 贊助商介紹詞

```
大家好，我是今年的贊助組組長品淳，接下來要進行的是由贊助商所組成的徵才活動。今年不論是對台灣或是對PyConTW都是非常動盪的一年，籌備團隊特別感謝贊助商的大力支持，在對第一次舉辦的線上會議成為我們最有力的靠山。接下來我將一一介紹贊助商並請請他們來對公司做深入的介紹
```

## 白金級國泰金控

```
- 第一個要介紹的是白金級的贊助商國泰金控
- 相信PyConTW 的大家對國泰金控都很熟悉，國泰金控連續三年最高級別贊助商，可以見得他們對於注社群，積極分享金融技術的重視，近年也開始舉辦他們的技術年會
- 在Podcast 熱潮之下也推出他們的podcast 芬泰食堂和大家分享金融科技的大小事
- 不論你是 大數據、資料科學、雲端技術、區塊鏈等技術專才，
只要對金融科技充滿熱情、都歡迎到攤位上找國泰的人員聊聊天喔
```

## 金級Kloudless

```
- 2018/2019 贊助商 提供 SaaS API 整合服務作為主要開發產品。Kloudless 於2021年初被Netskope 收購。
- 最近剛被一間美國的資安公司Netskope收購的 Kloudless Taiwan 科迪股份有限公司 ‼ 
- Netskope 則是一家位於美國資安公司。在世界各地共擁有 1500 多名員工，在併購後，Kloudless對於SaaS API 方面的技術與專業知識亦成功地運用在Netskope相關的雲端資安產品中。
```

## 銀級ICHEF

```
- 連續三年贊助商
- 平常去餐廳吃飯拿到的發票背後，很常看到 iCHEF 這個 Logo 吧
- iCHEF 是一間智慧餐飲科技公司，也是台灣iPad POS的領導品牌。巨量資料都是 iCHEF 團隊的重大挑戰
- iCHEF 資廚 是一個熱愛分享及切磋學習的研發團隊，因此每週各團隊都會有技術分享以及讀書會。
```

## KKBOX Group

```
KKBOX Group 除了大家耳熟能詳的 KKBOX 之外，集團跨足影音串流服務 KKTV、娛樂藝文活動展演 KKLIVE、行動售票及一站式虛擬場館直播平台 KKTIX、文創投資 KKFARM、串流技術 KKStream 等。

在許多技術社群裡我們也常常可以看到KKBOX Group 的工程師和大家分享串流相關的技術挑戰。
```

## 永豐金
```
- 首次贊助
- 永豐金證券於2019年推出Python API，
是迄今台灣累計交易金額最高的券商Python API套件。
- 使用者可以取得報價、下單、回報、帳務、歷史數據等服務。
- 永豐金在 Github、Telegram 等社群提供用戶交流討論，並持續更新完善功能與效能。
```

## BerryAI

```
- 首次贊助
- Berry AI 成立於 2019 年，專注於速食業 AI 應用，致力於運用電腦視覺技術幫助速食業者蒐集數據，改善現有營運流程，打造出更科技化的速食店。
- Berry AI 已與數間全球 Top-10 速食業者展開合作，業務與團隊都迅速擴張中，我們也在找尋更多充滿熱情，並喜愛尋求挑戰的夥伴加入!!
```

## Appier
```
- PyConTW 今年邁入第十年，很開心有 Appier 陪我們一起走過其中的八個年度，

- Appier 是一間透過人工智慧與軟體技術實踐行銷科技的公司。Appier 以AI與SaaS為核心，致力實現軟體智慧化讓 AI 更容易投入數位行銷產業。
- Appier 早期以預測廣告投放為主，如今 Appier 已具備圍繞顧客旅程AI 驅動解決方案，提供多樣化的行銷科技服務。
```


## OneDegree

```
- 今年的新贊助商
- 透過科技，改變傳統的保險生態從傳統保險產業推向另一個世代到純網路保險（Virual Insurance）

- 在金融領域，資訊安全是尤其重要的，我們觀察到市場對於資訊安全的需求，因此推出了資安即服務（Cybersecurity as a Service）平台，協助企業評估資安風險，提升資安曝險的可視性並以更敏捷及彈性化的方式管理資安風險。
```
