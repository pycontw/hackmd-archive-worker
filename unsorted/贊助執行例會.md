如果出面整合的人對於使用 github 和編輯 json 沒有障礙的話，和開發組與 opass team 合作上會省事一些（要怎麼改動>大地遊戲自己發 json pull request 改 configuration 就可以了）；不會改也沒關係，有辦法把需求說清楚讓別人發 pull request 也行
