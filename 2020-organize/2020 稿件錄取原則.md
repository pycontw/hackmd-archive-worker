# 2020 稿件錄取原則

## 錄取件數 (39場)
* 大會 (24 場)
    * 30 分鐘: 15 場
    * 15 分鐘: 9 場
* 社群軌 (15 場)
    * [AI/ML]: 3 場
    * [Finance]:  3 場
    * [Web/Data]: 3 場
    * [Core]: 3 場
    * [Science]: 3 場

## 錄取原則
1. 根據審稿指南的評分方式，如果未達直接拒絕，可以因為多樣性考量至少錄取一篇
    * **淘汰**：一個以上的 -1 沒有任何 +0、+1
    * **錄取**：一個以上的 +1 沒有任何 -0、-1
        * 100% 正評，60% 以上 +1
        * 100% 正評
2. 社群軌盡量按照主席規劃有各自屬性 （e.g., 每個主題三篇）
3. 不違背多樣性原則下，按照審稿指南的評分方式直接選取
    * 60%以上 +1
    * 80% 以上正評，且無 -1
4. 錄取稿件預留贊助已經確定的買下的 talk slot
5. valid review number 小於 3 的話，由議程主席裁決是否將評審評分視為有效

## 社群軌安排原則
* 避免 remote (但稿件不足的情況下可容許）
