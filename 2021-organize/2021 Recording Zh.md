# 錄影釋出

PyCon 致力於協助 Python 教育及提倡 Python 的使用，演講預設會由 PyCon Taiwan 委員會錄影、剪輯並上傳至線上影音網站（如：Youtube）以及 [pyvideo.org](http://pyvideo.org/)。在技術可行的情況下，我們也可能實況轉播議程的錄影。

如果您不希望自己的演講被錄影，請在投稿時取消勾選錄影的選項。
