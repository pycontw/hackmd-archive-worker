## 工作流程
1. requester 在下面新增「待廣播訊息」
2. 如果是比較緊急的廣播，請直接參考上述 PR 班表，直接在 discord 找人。
3. PR 組也會不定時審閱「待廣播訊息」
4. PR 組必要時會與 requester 協調、理解廣播脈落
5. PR 審核訊息無誤後，會去找有廣播管道的權限擁有者，請對方協助廣播。至此 PR 請更新「待廣播訊息」為「已處理」
6. 廣播權限擁有者執行廣播
